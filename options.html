      Add Domain

      Files stored on your computer by websites you've visited. These files contain user information, such as preferences for websites or profile information. — Source
    1. Clearing your browsing history specifically deletes the following: Web addresses you've visited from the History page, Cached text of those pages, Snapshots of those pages for images that appear on the New Tab page, Any IP addresses pre-fetched from those pages. — Source
    2. The list of files you've downloaded using Google Chrome is cleared, but the actual files are not removed from your computer. — Source
    3. All your Autofill entries and records of text you've entered on web forms are deleted. — Source
    4. Records of usernames and passwords are deleted. If you're using a Mac, your password records are deleted from Keychain Access. — Source
