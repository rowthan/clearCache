## WHY
这是一个缓存处理插件，主要以实现在chrome中缓存的处理功能。除此之外，为了更好的集成chrome API 相关操作，
以便自己和他人学习，重点针对以下内容进行了丰富。
* chrome 添加快捷键 API
* chrome 国际化处理
* chrome 插件通信 API
* HTML5 localStorage 存储。
* HTML5 拖拽功能。
